AUTOMATIC FAVICON GENERATION
==============================

SETUP (One-Time):
-----------------
1. Right-click logo in Figma Make → "Save Image As..."
2. Save to: /src/assets/302a78d8be4e75fe5f3bef65f80ada9b7aeb0688.png
   (or .jpg - plugin auto-detects format)
3. Run: npm run build

WHAT IT DOES:
--------------
✓ Extracts logo from src/assets/
✓ Handles base64 decoding if needed
✓ Auto-converts .png ↔ .jpg
✓ Generates all favicon sizes:
  - favicon.ico
  - favicon-16x16.png
  - favicon-32x32.png
  - apple-touch-icon.png (180x180)

FILES ARE ALREADY REFERENCED IN:
---------------------------------
/src/app/components/seo-head.tsx

CONFIGURATION:
--------------
Edit /vite-favicon-plugin.ts to change:
- App name
- Theme color (#0a3d3d)
- Background (#ffffff)
- Icon sizes

NO MANUAL WORK NEEDED - RUNS ON EVERY BUILD!
